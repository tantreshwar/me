College Projects




Rainwater Management System (Data Structure+IoT based+ DBMS):

The main concept is to operate the water pipe network,such that the area having less water can get water from area having sufficient amount.So here with Djikstra Algorithm we find the shortest path to tank where ample amount of water is there.


Intelligent Heating System (GUI based+IoT based+ DBMS+ Prediction Algorithm):

The main idea is that whenever a user goes to bathroom, he will find water heated perfectly to the temperature he likes. To operate remotely IoT concept is used.


Outbreak Alert System (IoT based+ Data Structure+DBMS+Comp.Networks):

This project is inspired from Covid-19 cases in India however it could be used for giving updates for any kind of outbreak.The data for patients is stored in form of hash table. From database using CSMA CD Non persistent protocol data is transmitted about outbreak via Internet and it messages user via Telegram.


Self-Balancing Bot (Control System based):

The project aimed at designing a bot that can balance itself on two wheels. The bot is wirelessly controlled using XBee module. The controller was implemented on Arduino.


Communication System (IoT based + Signal Processing):

The project aimed at implementing transfer of analog signal from electronic device via internet to receiver computer using digital communication. The programming language used was MATLAB.


Train Automation System (IoT based):

The project aimed at automation of Railway gates, indication of break in railway tracks and in emergency alarm will be raised at station in vicinity via internet also the signals will automatically turn red.


Temperature Control (IoT based):

The project aimed to monitor the temperature of the material using LM35 sensor. In case of out of range temperature the buzzer will switch on also message will be sent to monitoring station through internet.


Cricket Game (GUI based+DBMS):

It is a GUI application. The project aims at calculating the total point scored for selected players in a user defined team. Language used was Python and database management software used was MySQL.


Aircraft Reservation System (GUI based+DBMS):

It is a GUI application. The project aims at developing the booking and cancelling of flight tickets.Concept used is that a database is maintained of flights and tickets are booked, through which booking and cancelling is done.
